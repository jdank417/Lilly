Testing Instructions

To test the website, click the button below to open the main page in a new tab:

Open Website

Test the following features:

  1. Verify that the hero section loads with animations
  2. Test navigation links for smooth scrolling
  3. Check that the "Enter the Lab" button scrolls to the virtual lab section
  4. Test the microscope controls in the virtual lab:
    • Magnification slider should scale the yeast cells
    • Focus slider should adjust blur
    • Stain toggle button should change cell appearance
    • Time lapse button should speed up animations temporarily
  5. Verify that research section animations are working
  6. Test responsive design by resizing the browser window

Browser Compatibility

This website should work in all modern browsers including:

  • Google Chrome
  • Mozilla Firefox
  • Safari
  • Microsoft Edge

Some animations may not work in older browsers that don't support modern CSS and JavaScript features.
